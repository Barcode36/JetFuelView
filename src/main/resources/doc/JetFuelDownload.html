Please amend the StartJetFuel.bat file with the following
  1. Update JAVA_HOME to point to java 8 on your box
  2. Update AMPS_CONNECTION to be JSON connection url.  e.g. tcp://192.168.56.101:8001/amps/json (only json connection has been tested)
  3. Update AMPS_ADMIN_PORT to be admin port for the server e.g. 8199
  4. Update AMPS_ENV to be the environment, you can enter anything here like uat, dev, prod. If prod is entered you wont see the publish features of JetFuel.
Please note the following existing issues with this version:-
  • Sometimes the fields in the chooser panel does not get selected correctly. Or the validation method says no fields are selected. If this happens just right click and "Reset" the list or click on "Reset/Refresh Selection".
  • Only json connection string has been tested
  • New updates to show nested json is coming up.
